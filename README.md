# restChatCpp
A REST based web chat program using Node.js (express & socket.io)
## Requires node

## Set up (run from directory)
 - ```sudo mkdir /var/www/html/restChat/```
 - ```sudo chown ubuntu /var/www/html/restChat/```
 - ```sudo npm install```

## Make and run
 - ```make```
 - ```node ./restChat```
